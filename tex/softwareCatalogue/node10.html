Procedure

The latest version of ASPIC3 is version 7.01, there are two versions in the ICCAT catalogue i.e. 3.82 and 5.05; while the version on the NOAA Fisheries Toolbox is 5.34.9. There is also an R version developed under the GBYP (Merino et al., 2013).

The proposal is to

  1. Contact rapporteurs of assessment WG with a summary of the old requirements and additional issues that have arisen since the establishment of the Software Catalogue, e.g. related to the Strategic Plan, Kobe advice framework, SISAM/WCSAM, recent assessment and the use of stock assessment methods as part of a Management Procedures (MP) when conducting MSE. Ask rapporteurs to review if the old requirements are still adequate or need updating and to propose a set of revised requirements.
  2. Ask rapporteurs to use these new requirements to "certify" ASPIC.
  3. Canvassed views of software developers since if the process becomes too burdensome then no software will be developed.
  4. Canvassed views of other RFMOs.
  5. Presented results of the exercise to the SCRS which would approve a new protocol.
